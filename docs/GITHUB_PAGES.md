# GitHub Pages Notes

This project is currently designed for static GitHub Pages deployment from the repository root.

## Current setup

- `index.html` exists at the root
- CSS files are loaded from `styles/`
- Internal pages are stored in `pages/`
- `.nojekyll` is included
- No build step is required

## Local preview

Open `index.html` in a browser.

## Future deployment setting

In GitHub repository settings, Pages can be configured to serve from:

- Branch: `main`
- Folder: `/root`

After Sprint 1 is merged, the static entry point should be available for Pages.
