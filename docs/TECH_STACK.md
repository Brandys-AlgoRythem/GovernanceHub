# Tech Stack

## Sprint 1 confirmed stack

- Static HTML
- CSS
- No JavaScript required for Version 1 foundation
- Vanilla JavaScript planned for Version 2
- Mock JSON planned for Version 2

## Why this stack

Governance Hub is currently a portfolio prototype. Static files make the first version easy to review, easy to deploy, and easy to extend.

## Not included in Sprint 1

- React
- Next.js
- Vite
- Database
- Backend server
- Authentication
- External APIs
- Package manager
