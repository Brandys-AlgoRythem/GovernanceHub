# Accessibility Notes

Sprint 1 uses semantic HTML where practical.

## Current accessibility basics

- Pages include language attributes
- Pages include viewport metadata
- Navigation uses `aria-label`
- Main content uses `main`
- Metric groups use section labels where relevant

## Next improvements

- Add skip links
- Improve focus states
- Add stronger color contrast testing
- Add table captions when tables are introduced
- Add reduced motion support before adding animations
