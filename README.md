# Governance Hub

Governance Hub is a local-first governance operations portfolio prototype for small businesses.

The project demonstrates how obligations, evidence, risk, documentation, expenses, and audit readiness can connect inside one executive governance dashboard.

## Current sprint

Sprint 1 establishes the repository foundation.

Completed scope:

- Working branch: `print1`
- Static HTML/CSS architecture
- GitHub Pages-compatible entry point
- Foundational folder structure
- Starter landing page
- Starter dashboard and core page scaffolds
- Architecture, roadmap, and screenshot documentation

## Current stack

- HTML
- CSS
- Vanilla JavaScript planned for Version 2
- Mock JSON planned for Version 2
- No framework
- No package manager
- No backend
- No database

## Portfolio scope

Version 1 focuses on polished static pages and a CSS-heavy executive hive aesthetic.

Version 2 will add lightweight JavaScript using mock JSON data for dashboard metrics, due date states, search, filters, expense summaries, and relationship highlighting.

## Run locally

Open `index.html` in a browser.

Because this sprint does not use a package manager or build tool, there is no install step.

## GitHub Pages readiness

This project is designed to run as a static site from the repository root.
